Projet 1 :

Implémentation d'un besoin client :


Objectif : Algorithme capables de trier des dépêches.

Objectif : Java.

Equipe : Binôme.


- Le programme doit être capable de'analyser une base de donnée de 500 dépêches.

- Les dépêches sont séparées en 5 sujets possibles.

- Le programme doit apprendre à déterminer le sujet de chaque dépêches à partir d'une base de données exemple.

Compéten nécessaires : communication efficace, connaissance en java, travail d'équipe.
