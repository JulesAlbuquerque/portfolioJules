Projet 4 :

Création d'un site web :

Objectif : Créer un site web permettant à un élève de troisième cherchant un stage de s'informer facilement sur une entreprise du secteur du numérique.

Outils informatique : VSCode, Wireframe, Word.

Language : HTML, CSS, JS.

Equipe : Trinôme.

- Regrouper les informations concernant l'entreprise (CGI).

- Réaliser un receuil de besoin sur Word.

- Création d'un wireframe respectant les besoins établis.

- Création du site web en HTML, CSS, JAVA.

Compétences nécessaires : capacité à vulgariser, gestion des besoins, compétences en développement web, travail d'équipe.
